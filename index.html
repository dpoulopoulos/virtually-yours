Welcome to Virtually Yours!

Welcome to Virtually Yours, your go-to repository for creating Virtual Machines (VMs) using libvirt.

This repository aims to provide clear, easy-to-follow instructions for creating Windows and Linux VMs, along with configuration files to automate the process as much as possible.
